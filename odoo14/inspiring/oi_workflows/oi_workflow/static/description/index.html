        Description

        System Administrators and Business Managers will love this module!

        A workflow engine is a module or tool that manages business processes. It is a key component in any ERP systems. A workflow engine manages and monitors the state of activities in a workflow, such as the processing and approval of an employee contract and determines which new activity to transition to according to defined processes (workflows).

        Openinside’s Workflow Engine is a specially designed module to help users enforce a series of recurring tasks that make up a 'business process' or a 'workflow'. The Workflow Engine is a highly configurable module where almost no real coding is required to design any complicated processes.
