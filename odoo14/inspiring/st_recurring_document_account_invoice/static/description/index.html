Recurring Invoices

This module recreates Invoices in a time interval.

exemplary use cases:

  • The monthly supplier invoice for the Internet should be created automatically
  • The monthly customer invoice for a SLA Contract should be created automatically

The first invoice is created manually and the subsequent invoices are created automatically
