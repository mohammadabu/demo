Description

The sales sample product will help to manage sample product. We provide individual location for sample products. If there are sample products in the purchace order line, a receipt will be generated for the sample location otherwise receipt will be genereted for stock location. If there are sample products in the sale order line, a receipt will be generated from the sample location otherwise delivery will be genereted from stock location. You can configure, invoice include sample product qty or not.

Features List

  • Easy to view sample product stock.
  • Easy to create sale/purchase order line with sample product.
  • Create individual receipt/delivery for sample and normal product with different location.
  • You can configure, invoice include sample product qty or not.
