Sales Margin Report in PDF and EXCEL

Overview

This Odoo app will help to print Sales Margin Report in PDF & Excel. This Odoo app will help company management to measure progress in the sales and marketing section. In this Sales Margin Report, user can see a profit margin of each sales order. In this report, user can get all information like Sale Order Ref, Product Name, Product Cost Price, Product Sell Price, Discount, Tax, Untaxed Amount, and Margin.

Feature

  • User can easily analyze margin amount for the sales order and salesperson.
  • In this report, automated option to highlight negative margin and positive margin.
  • Use can generate PDF & EXCEL report.
  • Use can apply filters(For date range,salesperson,company) on large data.
  • Only Sales Manager can print this report.
  • This app is compatible with Odoo's Multi-Company Environment.
  • No more configuration is required, just install it and use it.

How It Works



Sales Margin Report Menu : Only Sales Manager(Sales Administrator) can see Sales Margin Report menu.




User can filter the data between two date rage, for selected companies and for selected salespersons.
