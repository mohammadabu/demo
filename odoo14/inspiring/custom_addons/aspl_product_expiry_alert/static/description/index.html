Odoo 14 Product Expiry Alert Dashboard (Community)

This module allows to product alert to going expire.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • This module allow to view all the product that are going to expire by location and warehouse.
  • you can search the location and warehouse that display all the products for that location or warehouse.
  • Also allow the show near expiry product graph to select custome date.
  • Allow to add near expire product blocks with configuration.
  • Send email of near expire product detail based on configuration.

How to use

  • In Inventory configuration setting, enable option "Variants", "Lots & Serial Numbers", "Expiration Dates", "Storage Locations" and "Send Mail".


  • After Enable all of this setting you need product that have enable lot or serial and mack sure Expiry date boolean is enable and if you want then give default Expiry day and default alert day.


  • After Configure product setting you need to do "Product Expiry Configuration" give number of day and box color and font color.


  • After configure all data and process if expiry date and alert date is available in lot or serial number it will display like this.








  • We add some color of line in tree view like yellow line is alert lot/serial and red line is expired lot/serial.
