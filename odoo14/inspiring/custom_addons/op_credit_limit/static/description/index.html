Customer Credit Limit

Adding Credit Limit for customers on sale orders with ability to give users access rights to hold it

Features

  • Adding Credit Control tap in Contact view
  • Users with Manager access rights can set value for credit limit
  • Users with Manager access rights can see summery of credit balance
  • Users with User access rights can only hold tjhe credit limit for the customer
  • sales users with users access right of credit limit can see how much balance and credit limit on sale order

  • users with credit limit access rights of user or manager can put credit limit on hold
  • all changes are stored in chatter
